# Returning the Gaze Exhibit

## First-time Setup

```shell
npm install
```

## Tips:

### Generate Favicon:
https://realfavicongenerator.net/

## Usage

### Live-Reloading Dev Server

```shell
npm run start
```

### Build Minified Deployment

```shell
npm run build
```

### Deploy to Github Pages

```shell
npm run build
npm run deploy
```
